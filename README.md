# Jim-E

## brief

The team built Jim‑E, an interactive AI‑assisted risk review tool that applies SME heuristics to project risk entries. Using a lightweight Streamlit interface and encoded heuristic rules, the solution helps users identify weak risks and mitigations, capture structured feedback, and generate clear audit‑ready reports.

## key outcomes

Improves the consistency and quality of risk register entries by systematically applying SME heuristics; reduces time spent reviewing risks and mitigations; provides clearer evidence trails to support earlier intervention and better risk decisions.

## important files

- `jim-e_ui_demo_final.py`: Streamlit user interface for capturing risk descriptions, SME agreement and feedback, and generating structured reviews.
- `hassans_brilliance.py`: Core application logic for managing submission history and producing PDF risk review reports.
- `Heurisitcs.docx`: Documented SME risk heuristics used to assess risk quality and mitigation strength.
